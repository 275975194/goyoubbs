# goyoubbs

goyoubbs is a web forum built on the Golang on fasthttp and leveldb.

Demo online https://youbbs.org/

## Usage

Require go 1.16+

Download source code and build.

```
git clone https://github.com/ego008/goyoubbs
cd goyoubbs
go build .
./goyoubbs
```
